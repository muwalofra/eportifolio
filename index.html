Launching into Computer Science

The overview Module for the MSc Program

Secure Software Development

This module explores security risks in programming languages, covering design, architecture, paradigms, testing, operating systems, libraries, distributed systems, APIs, and future trends

Object Oriented Programming

This module is about using python programming to support the design
and development of object oriented programs

Information Security Management

This module provided a comprehensive overview of information security management, covering key topics such as risk assessment, security standards, business continuity, and disaster recovery. Module outcomes include the ability to identify and manage security risks, analyze security breaches, and articulate the legal, social, ethical, and professional issues faced by information security professionals.

Software Engineering Project Management

This module looks at what it means to be a Project Manager of a software engineering project. It examines the relationships between the Project Manager and the other stakeholders in a project, and the role that the Project Manager has in ensuring that the triple constraints of project management are balanced, and any technical debt is not accrued. The module practically looks at how the needs of the customer are responded to through the use of test-driven development and behaviour-driven development.

Network Security

In this module we will look at the impact of cyber security on network systems and designs, focusing especially on network vulnerability assessments. We discussed techniques that can be used to scan, evaluate and in some case mitigate network vulnerabilities.

Research Methods and Professional Practice

This module is about practical research skills to enable us do research in real-life situations. It teaches different tools and methods we might need researchers and gives feedback through cases and projects. It also talks about important legal, social, ethical, and professional issues in computing projects, and talks about the rules that need to be followed.
